### Design Philosophy

> *I don't just use tools — I build them. Writing CPU emulators taught me what happens beneath every abstraction I rely on daily.*

- **Production-grade defaults:** Even academic projects get proper auth, caching layers, tenant isolation, and CI-ready structure.
- **Systems thinking:** Whether designing a three-bus CPU architecture or a multi-tenant SaaS backend — define clear interfaces, isolate concerns, make failure modes explicit.
